# Bench Folder Guide

## Scope
Benchmark and evaluation tooling (currently SWE-bench focused).

## Start Files
- `bench/swebench/bosun-swebench.mjs`

## Common Commands
- Import: `npm run bench:swebench:import`
- Export: `npm run bench:swebench:export`
- Evaluate: `npm run bench:swebench:eval`

## Notes
- Keep benchmark data and runtime logic isolated from production paths.
